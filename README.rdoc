= Mozilla Persona/BrowserID authentication plugin for Redmine

This plugin adds Mozilla Persona/BrowserID authentication support for Redmine.

== Requirements

Currently tested with ruby 1.9.x works with Redmine 3.4+

== Installation

    cd <redmine-install-dir>/plugins
    git clone https://github.com/martinsp/redmine_browserid.git

After installation restart redmine application
